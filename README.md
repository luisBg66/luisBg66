# Proyecto Laravel 🚀

Este repositorio contiene un proyecto basado en **Laravel**.

## 🚧 Requisitos
- PHP >= 8.1
- Composer
- MySQL / PostgreSQL
- Node.js & NPM

## ⚙️ Instalación
```bash
git clone https://github.com/tuusuario/tu-repo.git
cd tu-repo
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
npm install && npm run dev
